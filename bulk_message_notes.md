# 批量消息发送测试

## 对话框观察

批量发送消息对话框显示：
- 标题："批量发送消息"
- 副标题："向选中的用户发送消息通知"
- 用户选择区域：有 checkbox 列表，显示所有 20 个用户
  - 第一个 checkbox 是"全选 (20 个用户)"
  - 后面是各个用户的邮箱
- 类型选择：下拉框，当前选中"通知"
- 消息标题：输入框
- 消息内容：文本域
- 按钮："取消" 和 "发送给 0 个用户"

## 观察到的问题

1. 发送按钮显示"发送给 0 个用户"，说明还没有选择任何用户
2. 需要先选择用户，再填写内容，然后发送

## 测试步骤

1. 选择"全选"
2. 填写标题和内容
3. 点击发送

## 发送结果

批量消息发送成功！对话框关闭，回到用户列表页面。
没有看到明显的 toast 成功提示（可能是太快了）。
消息已发送给全部 20 个用户。
