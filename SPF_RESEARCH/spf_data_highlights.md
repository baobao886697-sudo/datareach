# SearchPeopleFree 数据亮点分析

## 实际测试结果

通过 scrape.do API 测试 John Smith 搜索，发现以下数据字段：

## 🌟 独家数据亮点（TPS/FPS没有）

### 1. 电子邮件信息
- 显示部分遮蔽的邮箱地址
- 示例：`jh*****r@epix.net`, `jh*****r@aol.com`
- 可显示多个邮箱地址

### 2. 婚姻状态与配偶信息
- 显示婚姻状态："Married to Jennifer A Smith"
- 配偶姓名带链接，可进一步查询

### 3. 电话类型标注
- **Home/LandLine Phone** - 座机
- **Wireless** - 手机
- 这是非常有价值的信息！

### 4. 就业状态
- 显示工作信息或 "[No known employment]"

### 5. 数据确认日期
- "Confirmed on January 27, 2026"
- 显示数据的新鲜度

### 6. 精确地理坐标
- 提供经纬度：`41.388416,-81.793122`
- 可用于地图定位和距离计算

### 7. 出生年份范围
- 显示 "Age 50 (1976 or 1975)"
- 比单纯年龄更精确

### 8. 地址居住时间
- 显示每个地址的报告时间
- 示例："reported in March 2008 (1 year)"

### 9. 关联人信息
- 家庭成员 (Family)
- 可能的关联人 (Possible Associates)
- 都带有链接可进一步查询

### 10. 企业关联
- 显示关联的企业信息
- 或 "No known businesses"

## 数据结构

### 详情页 URL 格式
```
https://www.searchpeoplefree.com/find/{firstname}-{lastname}/{unique_id}
```

### 搜索页 URL 格式
```
https://www.searchpeoplefree.com/find/{firstname}-{lastname}/{city}-{state}
```

## 开发建议

### 核心卖点
1. **电子邮件搜索** - 独家功能
2. **电话类型识别** - 区分座机/手机
3. **婚姻状态** - 独家数据
4. **数据新鲜度指标** - 确认日期

### 前端展示重点
- 突出显示电子邮件信息
- 电话号码旁标注类型（座机/手机图标）
- 显示数据确认日期
- 婚姻状态和配偶信息单独展示
